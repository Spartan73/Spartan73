SEARCH ENGINE OPTIMIZATION

How Search Engines Work

Search engines work by crawling hundreds of billions of pages using their own web crawlers. These web crawlers are commonly referred to as search engine bots or spiders. A search engine navigates the web by downloading web pages and following links on these pages to discover new pages that have been made available.

The Importance of Keywords and Descriptions as Meta Tags

Keywords, also known as Meta Tags, are snippets of code that tell search engines important information about your web page, such as how they should display it in search results. They also tell web browsers how to display it to visitors. For example, one of the very first keywords will begin with the Title Tag. Every web page has keywords, but they’re only visible in the HTML code.

The Four Elements that Comprise Writing Content Relevant Links

This list is a brief overview of four elements that expresses the importance of writing content relevant links.

  1. Links should bring qualified traffic to a website.
  2. Links should be strategically targeted for the audience.
  3. Links should come from authoritative pages.
  4. Links should come from topically relevant sopurces.

Some additional examples of implementing good relevant linking strategies include: Internal Linking, Inbound (External & Backlinks) Linking, Anchor Text and Nofollow Linking. Overall:

  • Good Internal Linking aids in website navigation, defines the architecture and hierarchy of the website and can properly distribute page authority and ranking throughout multiple pages of a website.
  • The use of Anchor Text helps Google understand what the link is about such as where you are linking and what the subject matter is.
  • Use relevant links that navigate the user to other areas of the website. For example; if the website you created is all about dress shoes and fashion trends, don’t provide a link to a website about automotive repair as the contents do not exactly corelate. Link to relevant content within your website.
  • Use links that are natural for the user. Doing this improves the overall user engagement and experience while navigating the website.

Good Heading Hierarchy

Headings should be logically structured on a page from biggest in text (most important) to smallest in text (least important). A page starts with the H1 heading, followed by an H2 heading and — should you need another sub-section — it can be followed by an H3 heading. If you need more sub-sections, you can even add H4-H6 subheadings. Again, from most important to least, top to bottom.

Hierarchy Examples

  1. h1 - A large heading should be short but describe the page's main topic
  2. h2 - Smaller subheadings describes a sub-category of information to follow
  3. h3 - An even-smaller subheading describes additional content and so on...
